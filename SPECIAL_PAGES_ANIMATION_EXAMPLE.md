## Technical Notes

### Animation Timing
- **Page 404**: 2.4 seconds per complete cycle (8 frames × 300ms)
- **Page 666 Animation 1**: 3.0 seconds per complete cycle (10 frames × 300ms)
- **Page 666 Animation 2**: 1.5 seconds per complete cycle (5 frames × 300ms)
- **Page 666 Animation 3**: 1.5 seconds per complete cycle (5 frames × 300ms)

### Synchronization
The three animations on page 666 run independently but create a harmonious effect:
- The demonic pulse is slowest (3s) - creates main focal point
- The pentagram and warning animations are faster (1.5s) - create ambient movement
- All animations loop continuously

### Maximum Effects
Both pages have `maxVisualEffects: true` which means:
- Animations play regardless of user settings
- Full glitch effects applied
- Chromatic aberration enabled
- Screen shake/flicker enabled
- No performance throttling

### Unique Features
- **Page 404**: Only page with broken TV glitch animation
- **Page 666**: Only page with triple simultaneous animations
- **Page 666**: Only page with AI-generated dynamic content
- Both pages: Only pages that override user animation preferences
