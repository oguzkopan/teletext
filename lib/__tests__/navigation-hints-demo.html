Implementation Notes

  • Automatic Detection: The system automatically detects page type and generates appropriate hints
  • Priority System: Custom hints (highest) → Context-specific → Default hints
  • Space Management: Hints are truncated if they exceed 40 characters
  • Color Support: Colored button hints include color metadata for styling
  • History Awareness: BACK hint only shown when navigation history exists
  • Page-Specific: Special hints for AI (loading), Quiz (progress), Settings (controls)

API Usage Examples

// Basic usage
const hints = generateNavigationHints(page, canGoBack);

// Selection page
const hints = generateSelectionHints(5);

// Content page
const hints = generateContentHints(true, false);

// AI page
const hints = generateAIHints(isLoading, canGoBack);

// Quiz page
const hints = generateQuizHints(3, 10, canGoBack);

// Error page
const hints = generateErrorHints();

// Index page
const hints = generateIndexHints();

// Render with layout engine
const rows = renderSingleColumn({
  pageNumber: '201',
  title: 'News',
  content: 'Article text...',
  hints: hints
});
