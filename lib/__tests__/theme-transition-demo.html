Usage Example

import { useTheme } from '@/lib/theme-context';

function ThemeSelector() {
  const { setTheme, isTransitioning } = useTheme();

  return (
    <button 
      onClick={() => setTheme('haunting')}
      disabled={isTransitioning}
    >
      Switch to Haunting Mode
    </button>
  );
}

Transition Phases

  1. Fade-Out (250-500ms) - Current theme fades out
  2. Switching (instant) - Theme state updated, Firestore save initiated
  3. Fade-In (250-500ms) - New theme fades in
  4. Banner (2 seconds) - Theme name banner slides in and out
